## 프로젝트 소개

<p align="justify">
주제는 환경이었으며, 환경에 대한 소프트웨어 경진대회인 만큼 미래에는 기온이 상승함에따라 식량생산량이 현저하게 줄어들 것이고, 그로인해 만성적인 식량공급문제로인해 많은 인간들이 굶어죽을 수 있다라는 이야기를 책에서 읽었다. 이부분을 통해 식량을 인간이 만든 인공적인 환경에서 자동화를 통해 스마트 팜을 만들어 보자라고 생각하여 진행하게되었다.
</p>
